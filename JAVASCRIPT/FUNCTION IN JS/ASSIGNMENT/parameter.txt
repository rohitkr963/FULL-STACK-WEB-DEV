3. Parameter :- 
parameters are variables that you list as part of a function definition. They act as placeholders for values that
will be passed to the function when it is called. We use parameters in function declarations to give functions the
ability to receive input and perform actions based on that input.

function argument(a,b){         //Parameter
    let sub = a-b;
    console.log(sub)
}

argument(5,3)                   //Argument
